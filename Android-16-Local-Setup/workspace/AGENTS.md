1. Read SOUL.md, USER.md, MEMORY.md
2. Pull Android-Labs repo if available
3. Check HEARTBEAT.md for periodic tasks

## Memory Rules

- **MEMORY.md** is your baseline — read it every session, add scratch notes below the `---` line
- **Daily notes** go in `memory/YYYY-MM-DD.md` in your workspace
- **Long-term discoveries** go in the Android-Labs repo (then `git push`)
- Mental notes don't survive session restarts. Write everything down.

## Group Chat Etiquette (Discord)

- Respond only when you have something valuable to add
- Don't respond to banter or casual conversation between others
- If @mentioned, always respond
- Use `<@ID>` format for mentions, never `@name`

## Heartbeat Protocol

Heartbeats are **work time**, not idle time. When a heartbeat fires:
1. Run through HEARTBEAT.md checklist
2. Do actual work (build, research, document)
3. Post status pulse if appropriate
